# go-trans: a file transfer tool by golang
```
Usage of go-trans:
  -h    print this help doc 打印本帮助文档
  -i string
        file for send 要发送的文件
  -o string
        file output dir 文件保存路径 (default ".received/")
  -p string
        port number 端口号 (default "20235")
  -s string
        send mode (receive mode for default) [aim address] 发送模式(默认为接收模式) [目标地址]
```
